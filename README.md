# 3D 粒子手势交互系统

## 1. 系统简介

本项目是一个基于 Web 的 3D 视觉应用。它通过电脑摄像头捕捉您的手部动作，实时控制屏幕上由基础 25,000 个光点组成的 3D 模型。

- **无需鼠标**：完全通过手势操作。
- **实时响应**：移动手掌可旋转模型，捏合手指可让粒子扩散爆炸。
- **高度可玩**：支持多种模型切换、颜色流光特效及参数微调。

------



## 2. 运行环境要求

为了获得最佳体验，请确保您的设备满足以下条件：

- **硬件**：带有摄像头的台式机或笔记本电脑（建议配备独立显卡以获得流畅帧率）。
- **浏览器**：最新版本的 Chrome、Edge、Firefox 或 Safari。
- **网络**：首次运行需要联网加载 AI 模型和 3D 引擎库。

## 4. 手势交互说明

系统启动后，浏览器左上角会询问**摄像头权限**，请务必点击 **“允许”**。

### ✋ 1. 激活控制

- 将一只手（左手或右手均可）举起，置于摄像头画面中央。
- 左上角状态栏显示绿色文字 **“状态: 交互中”** 时，表示系统已锁定您的手势。

### 🔄 2. 旋转模型 (Rotation)

- **操作**：在空中上下左右移动您的手掌。
- **效果**：模型会跟随您的手掌移动方向进行旋转。
  - 手向**右**移 -> 模型向**右**自转。
  - 手向**上**移 -> 模型向**上**翻转。
- *设计理念：如同您隔空“抓”着物体在转动。*

### 💥 3. 粒子扩散 (Expansion)

- **操作**：控制 **食指** 与 **拇指** 的开合距离。
- **效果**：
  - **五指张开**（或食指拇指分开）：粒子受到能量冲击，向外**炸开/扩散**，体积变大，形状变散。
  - **捏合手指**（OK手势）：粒子受到引力吸引，**聚合**成紧致的实体模型。

## 5. 控制面板功能详解

屏幕右上角有一个悬浮的控制面板，用于自定义视觉效果。

### 📁 模型 (Shape)

- **当前形状**：下拉选择不同的 3D 结构。
  - ❤️ 饱满爱心：核心主打形状，体积感强。
  - 🧬 DNA：动态旋转的双螺旋结构。
  - 🗿 佛像：程序化生成的禅意造型。
  - 🌌 银河 / 🪐 土星 / 🍩 扭结 等。
- **粒子大小**：调整光点的粗细（建议值 0.10 - 0.20）。

### 📁 颜色 (Color)

- **颜色模式**：
  - 固定：显示单一颜色。
  - 幻彩：开启 RGB 流光特效，颜色随时间和空间流动。
- **拾色器**：在固定模式下，点击色块可自由选取颜色。
- **流光速度**：控制幻彩模式下颜色变幻的快慢。

### 📁 交互参数 (Interaction)

- **旋转力度**：数值越大，手部微小的移动就能引起模型大幅度的旋转（建议 5.0 - 15.0）。
- **扩散力度**：数值越大，张开手指时粒子炸开的范围越广。
- **平滑系数**：
  - 数值越小（如 0.05）：动作越有“惯性”和“延迟感”，显得飘逸。
  - 数值越大（如 0.20）：响应越灵敏跟手。
- **摄像头画面**：勾选后可在左下角看到实时的手势捕捉画面（方便调试）。

## 6. 常见问题排查 (FAQ)

**Q: 打开后一直显示“Loading / 初始化中”，进度条不动？**

- **A**: 请检查网络连接（需要下载 AI 模型）。如果网络正常，请按 F12 打开开发者工具，查看 Console 是否有红色报错。通常是因为没有使用本地服务器（Local Server）运行导致的。

**Q: 画面是黑的，或者没有反应？**

- **A**: 请确认您已点击浏览器地址栏左侧的图标，授予了摄像头权限。如果还是不行，尝试刷新页面或更换 Chrome 浏览器。

**Q: 只有圆点在转，手势没反应？**

- **A**: 请确保环境光线充足，不要背光。将手掌正对摄像头，保持在距离摄像头 30-50cm 处。

**Q: 电脑运行很卡顿，风扇狂转？**

- **A**: 该项目渲染了 25,000 个粒子，对显卡有一定要求。您可以尝试在控制面板中减少粒子数量。
